# Sim2Real — NeurIPS 2026 Workshop companion site

A single-page static site (no build step). Companion to the workshop proposal.

## Files
- `index.html` — the page (all content lives here)
- `styles.css` — styles (Sim = blue `#2563eb`, Real = deep green `#15803d`)

## Preview locally
Open `index.html` in any browser (keep `styles.css` in the same folder).

## Deploy to GitHub Pages
1. Create a repo named **`sim2real-neurips26.github.io`** (under the org/user that owns the URL).
2. Push `index.html` and `styles.css` to the `main` branch (repo root).
3. Settings → Pages → Source: `main` / root. The site goes live at
   `https://sim2real-neurips26.github.io/`.
